== Description ==

Allows the user to monitor backlinks, alexa rank, and Facebook share right from the user's Wordpress Dashboard.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View your Wordpress Dashboard to see your stats

== Frequently Asked Questions ==

 = Why is it called Dashboard Social Stats? =

Because the most likely use of this plugin is for those who are looking to track statistics related to their site's social reputation.
